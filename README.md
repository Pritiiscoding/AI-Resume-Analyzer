# AI Resume Analyzer & Interview Assistant

## Overview

This project is an AI-powered Resume Analyzer and Interview Assistant.

It helps users:

- Upload resumes
- Analyze ATS compatibility
- Identify missing skills
- Suggest improvements
- Generate interview questions
- Conduct AI-based mock interviews

---

## Tech Stack

- Python
- FastAPI
- NLP
- Sentence Transformers
- LLM APIs
- FAISS / ChromaDB
- Streamlit (later)
- Git & GitHub
